SK8

Una pagina web hecha para una skateshop, en este proyecto se busco una interfaz minimalista, moderna y funcional, en esta misma se encuentran todos los productos de la tienda, incluyendo imagen, precio, descripción y stock. Tambien cuenta con un backend, donde se lleva el control de inventario, de los pedidos y de las ventas. todo esto se encuentra en una base de datos mysql, la cual se conecta con el backend para poder llevar un control de todo lo que sucede en la tienda.

Tecnologías PHP, HTML, CSS, JavaScript, MySQL
Lo que aporta Una solución práctica para la gestión de una skateshop, con una interfaz intuitiva y un backend robusto.
