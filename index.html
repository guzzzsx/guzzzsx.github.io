Proyectos

SK8

Una pagina web hecha para una skateshop, en éste proyecto se buscó una interfaz minimalista, moderna y funcional, en esta misma se encuentran todos los productos de la tienda, incluyendo imagen, precio, descripción y stock. Tambien cuenta con un backend, donde se lleva el control de inventario, los pedidos y las ventas. Todo esto se encuentra en una base de datos mysql, la cual se conecta con el backend para poder llevar un control detallado y preciso.

Abarrotes Belén

Un programa/sistema de gestión para una tienda de abarrotes que buscaba optimizar su operación y mejorar la experiencia del cliente. Éste programa permitió automatizar procesos y aumentar la eficiencia operativa. Todo esto de la mano de una apariencia amigable e intuitiva que solicitó el cliente.

Azuu Biblioteca
